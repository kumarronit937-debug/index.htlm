# Project Description
# Personal Portfolio Website
--------------------------------------

## 1. Project Overview
This is a beginner portfolio website created using HTML.
It contains:
- Header and navigation
- About section
- Skills section
- Contact form
- Images and external links

Purpose:
To practice basic HTML concepts and webpage structure.

## 2. Setup Instructions
---------------------------------

1. Download or clone the project
2. Open folder in VS Code
3. Open index.html in browser
